<h1 align="center">Drawoza</h1>

<p align="center">
  Draw anything, simply.
</p>

Simple web drawing app made with HTML, CSS, and JavaScript.

![Drawoza preview](images/preview.png)

## Features

- Pencil tool
- Move tool
- Line tool
- Rectangle tool
- Circle tool
- Eraser
- Text tool
- Color picker
- Brush size control
- Zoom controls
- Undo and redo
- Clean simple interface

## Tech Stack

- HTML
- CSS
- JavaScript
- Canvas API
